## LeanSeed

a seed project for LeanEngine.

### Init Project
```bash
$ git clone git@github.com:guo-yu/leanengine-seed.git
$ cd leanengine-seed
$ cnpm install .
$ avoscloud add <appName> <appId>
```

### Dev
```bash
$ avoscloud
```
then visit `http://localhost:3000`

### Deploy to Dev environment
```bash
avoscloud deploy
```

### Publish to production
```bash
avoscloud publish
```
